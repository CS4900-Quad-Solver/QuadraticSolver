Finds all real roots from given coefficients.

Input and output is limited to IEEE-binary32 or single-precision floating point.


changes:Split up project into separate c files to facilitate easier unit testing and individual improvement of functions.

Integrated unit testing framework authored by John Kapenga and successfully conducted unit tests.

This variation may not be better, but just easier to work with testing, but hopefully it is at least equivalent.



compile with 0 warnings or errors using GCC 5.4.0 on a Windows box running Cygwin, have not tested with cs machines.
